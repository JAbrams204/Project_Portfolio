Dice Game

Online Dice Game

Click for Demo Repository

This application was built using HTML, CSS and plain JavaScript. It is an Online Dice Game designed for two players. Each player takes turn to roll the dice and the first player to amass a cumulative score of twenty or greater wins the game.

In creating this application, several JavaScript concepts were used such as Bespoke functions, Built-in functions, If else statements, Variables, CSS custom properties, Flex-box, Classes, DOM nodes, Buttons, Event listeners and Random number generator.
